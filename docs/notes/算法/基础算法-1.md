# 基础算法-1

## 快速排序 —— 分冶

1. 确定分界点 ：可以选择左边界`q[l]`、中间点`q[(l+r)/2]`、右边界`q[r]`，也可以随机取。

2. 调整区间，将区间分为`<=x`和`>=x`两部分。

3. 递归处理左右两段。

代码模板如下：

```C++
void quick_sort(int q[], int l, int r)
{
    if (l >= r) return;

    int i = l - 1, j = r + 1, x = q[l + r >> 1];
    while (i < j)
    {
        do i ++ ; while (q[i] < x);
        do j -- ; while (q[j] > x);
        if (i < j) swap(q[i], q[j]);
    }
    quick_sort(q, l, j), quick_sort(q, j + 1, r);
}
```
